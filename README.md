# Lightweight Model Project

## Overview
This project is focused on developing a lightweight machine learning model for efficient performance and accuracy.

## Project Structure
```
/D:/Projects/Lightweight model/Test_1/
├── data/               # Dataset files
├── models/             # Trained models
├── notebooks/          # Jupyter notebooks for experiments
├── src/                # Source code
├── tests/              # Unit tests
└── README.md           # Project documentation
```

## Installation
To install the required dependencies, run:
```bash
pip install -r requirements.txt
```

## Usage
To train the model, use the following command:
```bash
python src/train.py
```
